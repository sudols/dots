# Dotfiles

My Hyprland configuration files for Arch Linux.

## Contents

- **hypr/** - Hyprland window manager config
- **waybar/** - Status bar config
- **kitty/** - Kitty terminal config
- **rofi/** - Application launcher config
- **alacritty/** - Alacritty terminal config

## Installation

```bash
# Clone the repo
git clone git@github.com:YOUR_USERNAME/dots.git ~/dev/dots

# Copy configs to ~/.config/
cp -r ~/dev/dots/hypr ~/.config/
cp -r ~/dev/dots/waybar ~/.config/
cp -r ~/dev/dots/kitty ~/.config/
cp -r ~/dev/dots/rofi ~/.config/
cp -r ~/dev/dots/alacritty ~/.config/
```

## Dependencies

- hyprland
- waybar
- kitty
- rofi
- alacritty (optional)
